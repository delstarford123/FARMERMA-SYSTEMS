Farmerman Systems API

Build agricultural intelligence into your own applications. The Farmerman API allows Enterprise clients to fetch live commodity prices directly from our Kakamega and Western region data nodes.

Authentication

The Farmerman API uses Bearer Tokens to authenticate requests. You must include your Enterprise API key in the `Authorization` header of every request.

Keep your token secure. Never expose it in client-side code (like frontend JavaScript). Route all requests through your secure backend.

Retrieve Market Prices

Fetch an array of live agricultural commodity prices across tracked regions.

GET https://farmerman-systems.onrender.com/api/market-prices

Error Codes

401 Unauthorized No API token provided, or token is invalid.
403 Forbidden Valid token provided, but your account is not on the Enterprise tier.
500 Server Error An internal issue occurred connecting to the Farmerman Database.
